Sintaxis de iframe HTML ⛲

La etiqueta IFRAME, se utiliza como un marco en línea para incrustar otro documento dentro del documento HTML actual.

Sintaxis iframe src="url" title="description"

Es una buena práctica incluir siempre un title atributo para IFRAME. Los lectores de pantalla lo usan para leer cuál es el contenido del iframe...

Establecer altura y ancho ⛄

Para esto vamos a usar los atributos HEIGTH y WIDTH para especificar el tamaño del iframe.

🌊 La altura y el ancho se especifican en píxeles de forma predeterminada.

🌊 O podemos agregar el style atributo y usar el CSS heighty las width propiedades

🌊 Eliminar el borde o agregarle estilos

🌝 Iframe: objetivo de un enlace

Se puede usar un iframe como marco objetivo para un enlace.
El target atributo del enlace debe referirse al name atributo del iframe
