ELEMENTOS DE LINEA Y DE BLOQUE ✊

Cada elemento HTML tiene un valor de visualización predeterminado, según el tipo de elemento que sea. Hay dos valores de visualización: bloque e inline.

Elementos a nivel de bloque

Un elemento de nivel de bloque siempre comienza en una nueva línea y ocupa todo el ancho disponible (se extiende hacia la izquierda y hacia la derecha tanto como sea posible).

El elemento DIV es un elemento de nivel de bloque.

El elemento DIV y SPAN 🌛

El elemento DIV es un elemento de bloque, se usa a menudo como contenedor para otros elementos HTML tambien de bloque o de linea.
Las etiquetas DIV y SPAN, no tienen ningún tipo de significado especial, solo que la primera se define como un elemento de bloque y por tanto el navegador mostrara un salto de línea antes y después de la misma. Mientras que la segunda es un elemento de línea.

Que onda con SPAN 🌠

La etiqueta SPAN también funciona como un contenedor pero en este caso será un contenedor en línea. Para destacar una palabra dentro de un párrafo utilizaremos la etiqueta SPAN, para encerrar esa palabra o expresión y luego ponerle un estilo mediante un atributo. O también podemos ponerle un id o class para luego maquetarlo con CSS.

Este es un párrafo en el que insertaremos un elemento span a continuación para diferenciarlo Texto especial dentro de etiqueta span con fondo amarillo y color rojo ese texto especial sigue dentro del párrafo y en linea, forma parte del párrafo, pero podemos especificar alguna característica

🌱 En resumen, la diferencia entre el div y span es que div se emplea para agrupar elementos html en un bloque en cambio span sirve para agrupar dentro de una línea.
